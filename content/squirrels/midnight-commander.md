## Installation from GitHub

1. Download [`squirrelsong-dark.ini`](https://github.com/sapegin/squirrelsong/raw/master/themes/Midnight%20Commander/squirrelsong-dark.ini), and place it in `~/.local/share/mc/skins`.
2. Open **Settings** menu, then **Appearance**.
3. Select `squirrelsong-dark` in the **Skin** dropdown.
