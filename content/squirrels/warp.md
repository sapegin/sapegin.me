## Installation from GitHub

1. To start, create the following directory: `mkdir -p ~/.warp/themes/`.

**Note:** It may take several minutes for Warp to initially discover the new themes directory. You can either wait or restart Warp.

2. Copy the [`squirrelsong_dark.yaml`](https://github.com/sapegin/squirrelsong/raw/master/themes/Warp/squirrelsong_dark.yaml) to this directory.
3. Open **Settings**, then **Appearance**.
4. Press **Current theme** box under **Themes** section.
5. Select **Squirrelsong Dark** in **Themes** list.
